# 動画圧縮ツール

動画を軽くするツール。H.265(HEVC)でエンコードし、ファイルサイズを削減する。

## 行うこと

- H.265(libx265)でのエンコード
- 音声トラックの保持
- メタデータの削除

## エンコード設定

| 設定             | 値      |
| ---------------- | ------- |
| 映像コーデック   | libx265 |
| プリセット       | slow    |
| CRF              | 20      |
| 音声ビットレート | 128k    |

## 必要なツール

- Deno
- Node.js
- FFmpeg

## 使い方

### 開発実行

```sh
deno task dev
```

### ビルド

```sh
deno task build
```

実行ファイルが `./dist/video-compression` に生成される。

### 実行

起動するとパスの入力を求められる。ファイルパスまたはディレクトリパスを入力する。

- **ファイル指定**: 指定したファイル単体を変換する
- **ディレクトリ指定**: `.mp4` / `.mov` / `.avi` / `.mkv` を一括変換する

出力ファイルは元のファイルと同じディレクトリに `<元のファイル名>-convert.mp4` として保存される。
